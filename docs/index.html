Project Myriad

The Definitive Manga & Anime Platform

A comprehensive Kotlin Android application featuring AI-powered tools, local media management, and seamless online content discovery built with modern Android architecture.

Powerful Features

Everything you need for the ultimate manga and anime experience

📚

The Vault

Local media management with support for .cbz/.cbr manga archives and video files. Organize your collection with smart categorization and metadata extraction.

🤖

AI Core

AI-powered OCR translation, art style matching, and intelligent recommendations. Get real-time manga translation and personalized content suggestions.

🌐

The Browser

Extensible online content discovery with plugin architecture. Search multiple sources simultaneously and manage downloads efficiently.

🎨

Enhanced Reader

Multiple reading modes including single page, double page, continuous scroll, and webtoon. Customizable interface with themes and layout preferences.

📊

Progress Tracking

Detailed reading statistics, completion tracking, and personalized insights. Never lose track of your favorite series again.

🏷️

Smart Organization

Custom collections, tags, and automated library organization. Find your content quickly with powerful search and filtering.

Modern Technology Stack

Built with cutting-edge Android development tools

🏗️ Core

  • Kotlin - Modern, expressive language
  • Android SDK 36 - Latest capabilities
  • Material 3 - Modern design system
  • Jetpack Compose - Declarative UI

🏛️ Architecture

  • Clean Architecture - Separation of concerns
  • MVVM Pattern - Reactive UI architecture
  • Hilt - Dependency injection
  • Room - Local database persistence

🌐 Networking

  • Retrofit - Type-safe HTTP client
  • OkHttp - Efficient network operations
  • Kotlinx Serialization - JSON parsing
  • Coroutines - Asynchronous programming

🧪 Quality

  • JUnit - Unit testing framework
  • Espresso - UI testing
  • Mockk - Mocking framework
  • Detekt - Code quality analysis
