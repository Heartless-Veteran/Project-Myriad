## 🌟 Key Features
📚

The Vault

Local media management system for .cbz/.cbr manga files and .mp4/.mkv/.avi anime videos with advanced organization.

🤖

AI Core

Intelligent OCR translation, art style matching, and AI-powered content recommendations powered by Google ML Kit.

🌐

The Browser

Extensible online content discovery with support for multiple manga and anime sources through plugin architecture.

🎨

Modern UI

Beautiful interface built with Jetpack Compose and Material Design 3, offering smooth animations and responsive design.
